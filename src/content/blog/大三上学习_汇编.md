### 寻址方法
一般说的有效地址为内存中的地址

1. 立即数寻址：无有效地址，操作数在指令中
    ```assembly
    MOV AX, 637DH ; 将立即数 637DH 加载到 AX 寄存器中
    ADD SI, 0100H ; 将立即数 0100H 加到 SI 寄存器中
    ```
2. 直接寻址：有效地址 EA = 偏移量 D = 7237H
    ```assembly
    MOV AX, [7237H] ; 将内存地址 7237H 处的内容加载到 AX 寄存器中
    ADD SI, [7237H] ; 将内存地址 7237H 处的内容加到 SI 寄存器中
    ```
    直接寻址必须强制指明段寄存器，哪怕就是默认的DS

3. 寄存器寻址：EA 不适用，操作数在寄存器中
   - 任何操作数不能是IP、FLAGS，目的操作数不能是CS。
    ```assembly
    MOV AX, BX ; 将 BX 寄存器的内容加载到 AX 寄存器中
    ADD SI, BX ; 将 BX 寄存器的内容加到 SI 寄存器中
    ```

4. 寄存器间接寻址：EA = (寄存器)
   - 该寄存器必须是BX、SI、DI、BP中的一个，其它寄存器不可以。
   - 如果寄存器是BX、SI、DI，默认的段寄存器是DS。
   - 如果寄存器是BP，默认的段寄存器是SS。
    ```assembly
    MOV AX, [BX] ; 将 BX 寄存器指向的内存地址处的内容加载到 AX 寄存器中
    ADD SI, [BX] ; 将 BX 寄存器指向的内存地址处的内容加到 SI 寄存器中
    ``` 

5. 寄存器相对寻址：EA = (寄存器) + D
    ```assembly
    MOV AL, [BX-45H] ; 将 BX 寄存器的内容减去 45H 后指向的内存地址处的内容加载到 AL 寄存器中
    ADD [SI+1050H], 0102H ; 将 SI 寄存器的内容加上 1050H 后指向的内存地址处的内容加上 0102H
    ```

6. 基址变址寻址：EA = (寄存器1) + (寄存器2)
    - 间接寻址的寄存器限制在此仍然适用，基址寄存器只要是 BP，那么默认的段寄存器就是SS，否则是DS。还要额外施加限制：两个寄存器必须分别是基址寄存器（BP或BX）和变址寄存器（SI和DI）。
    ```assembly
    MOV AX, [BX + SI] ; 将 BX 寄存器和 SI 寄存器的内容相加后指向的内存地址处的内容加载到 AX 寄存器中
    ADD SI, [BX + SI] ; 将 BX 寄存器和 SI 寄存器的内容相加后指向的内存地址处的内容加到 SI 寄存器中
    ```

7. 相对基址变址寻址：EA = (寄存器1) + (寄存器2) + D
    ```assembly
    MOV AL, [BX+SI-45H] ; 将 BX 和 SI 的内容相加后减去 45H 指向的内存地址处的内容加载到 AL 寄存器中
    ADD [BP+DI+1050H], 0102H ; 将 BX 和 SI 的内容相加再加上 1050H 后指向的内存地址处的内容加上 0102H
    ```

#### 数据传送指令
**MOV**
`MOV target, source` 将 source 的内容复制到 target 中 
target : 寄存器、存储器、段寄存器
source : 寄存器、存储器、段寄存器、立即数
- 无法移动立即数到段寄存器
- 无存储器到存储器的操作类型
- 无段寄存器到段存储器的操作类型
- target 和 sourse 数据类型需要匹配

**PUSH**
`PUSH source` 将 source 的内容压入堆栈
source : 仅接受字操作，不可以使用 IP、FLAGS

**POP**
`POP target` 将堆栈顶的内容弹出到 target 中
target : 仅接受字操作，不可以使用 CS、IP、FLAGS

**XCHG**
`XCHG target, source` 将 target 和 source 的内容交换
target : 寄存器、存储器
source : 寄存器、存储器
交换的两个寄存器的字长必须一样
LOCK XCHG : 上自旋锁

**XLAT** 不要求
AL <= DS:[BX + AL]

**LEA**
加载有效地址
`LEA target, source` 将 source 的有效地址加载到 target 中
例子：`LEA SI, [BP + SI + 8241H]`
target : reg16
source : [mem] 和 MOV 不同，reg 会变成 mem（地址）

# PPT第10讲

- **中断号**：决定了**向谁请求服务**
    - `INT 10H`, `13H`, `14H`, `16H` → **BIOS服务**（底层硬件）
    - `INT 21H` → **DOS服务**（操作系统）
    - `INT 33H` → 鼠标服务
- **功能号**：决定了**请求什么具体的服务**
    - 放在 `AH` 寄存器中
    - **相同的功能号在不同中断下含义完全不同**
