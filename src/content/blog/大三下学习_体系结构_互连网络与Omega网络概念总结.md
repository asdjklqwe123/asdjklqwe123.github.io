# 互连网络与 Omega 网络概念总结

## 1. 互连函数

互连函数描述输入到输出的映射 $f(i)=j$（N 个结点）：

| 函数 | 定义 | 例子 (N=8) |
|:---|:---|:---|
| **恒等** | $f(x)=x$ | 3→3 |
| **Cubeₖ** | $f(x)=x \oplus 2^k$（第 k 位取反） | Cube₁(3)=3⊕2=1 |
| **PM2I₊ᵢ** | $f(x)=(x+2^i) \bmod N$ | PM2₊₁(3)=(3+2) mod 8=5 |
| **PM2I₋ᵢ** | $f(x)=(x-2^i) \bmod N$ | PM2₋₁(3)=(3-2) mod 8=1 |
| **均匀洗牌 σ** | 二进制左循环移一位 | σ(3)=σ(011)=110=6 |
| **逆洗牌 σ⁻¹** | 二进制右循环移一位 | σ⁻¹(3)=σ⁻¹(011)=101=5 |
| **蝶式 β** | MSB 与 LSB 互换 | β(3)=β(011)=110=6 |
| **位反转 ρ** | 全部位序反转 | ρ(3)=ρ(011)=110=6 |

---

## 2. 均匀洗牌（Perfect Shuffle）

**概念**：像洗扑克牌一样交叉排列，二进制上就是**循环左移一位**。

**例子 (N=8)**：
```
上半 0 1 2 3  → 交错后 →  0 4 1 5 2 6 3 7
下半 4 5 6 7
```

二进制验证：σ(1)=σ(001)=010=2，σ(4)=σ(100)=001=1。

---

## 3. 交换开关（SE）

**概念**：2×2 开关，有四种状态，但点对点通信只用两种：

| 状态 | 连线 | 条件 |
|:---|:---|:---|
| **直通 S** | 上入→上出，下入→下出 | 上入目标位=0，下入目标位=1 |
| **交叉 C** | 上入→下出，下入→上出 | 上入目标位=1，下入目标位=0 |

**冲突**：两输入要求同一输出端口时发生。即两输入的目标控位相同（都=0 或都=1）。

**例子**：上入目标位=0，下入目标位=0 → 都要求走上出 → 冲突！

---

## 4. Omega 网络拓扑

**概念**：$\log_2 N$ 级 SE，级间用均匀洗牌连接。术语：
- **第 0 级**：输入处理器直连的第一级 SE（还没经过任何洗牌）
- **洗牌**：上一级 SE 的输出线经过均匀洗牌重排，变成下一级 SE 的输入
- **第 1 级**：经过一次洗牌后的 SE
- **第 2 级**：经过两次洗牌后的 SE（直连输出处理器）

即：`输入 → 第0级SE → 洗牌 → 第1级SE → 洗牌 → 第2级SE → 输出`

两条固定规则：
- **规则 A**：每级相邻两线绑进一个 SE
- **规则 B**：级间连线 = 均匀洗牌

**例子 (N=8)**：
```
输入处理器        第0级SE         第1级SE          第2级SE        输出处理器
   0  ────[SE₀]───洗牌───[SE₀]───洗牌───[SE₀]──── 0
   1  ────┘              ───┘              ───┘      1
   2  ────[SE₁]──────────[SE₁]──────────[SE₁]──── 2
   3  ────┘              ───┘              ───┘      3
   4  ────[SE₂]──────────[SE₂]──────────[SE₂]──── 4
   5  ────┘              ───┘              ───┘      5
   6  ────[SE₃]──────────[SE₃]──────────[SE₃]──── 6
   7  ────┘              ───┘              ───┘      7
```

- 第 0 级 SE 配对：(0,1)(2,3)(4,5)(6,7) — 输入直连，相邻即相邻处理器。
- 第 1 级 SE 配对：洗牌后变成 (0,4)(1,5)(2,6)(3,7) — 规则没变（仍是相邻线进同一 SE），是线上的人被洗牌换了位置。

---

## 5. 目的地标号路由

**概念**：从输入到目标输出，每级由目标地址的一位控制走线上还是线下。

| 级 | 控制位 | 0 | 1 |
|:---:|:---|:---|:---|
| 第 0 级 | MSB (bit n-1) | 走上出 | 走下出 |
| 第 1 级 | bit (n-2) | 走上出 | 走下出 |
| 第 2 级 | LSB (bit 0) | 走上出 | 走下出 |

**例子 (N=8，从输入 0 到输出 6)**：目标 6 = 110₂
- 第 0 级：MSB=1 → 走下出
- 第 1 级：bit1=1 → 走下出
- 第 2 级：LSB=0 → 走上出

---

## 6. SE 状态判定

**概念**：查同一 SE 两个输入的目标控位：

| 上入控位 | 下入控位 | SE 状态 |
|:---:|:---:|:---:|
| 0 | 1 | 直通 S |
| 1 | 0 | 交叉 C |
| 0 | 0 | 冲突 |
| 1 | 1 | 冲突 |

**例子**：SE 收到输入 2（目标=010，当前级控位=0）和输入 4（目标=001，当前级控位=0）→ 都是 0 → 冲突！

---

## 7. 冲突判断流程

**概念**：逐级检查。任一级有冲突 → 该置换不能一次通过 Omega 网络。

**例子 — 位反转 β(abc)=cba (N=8)**：
- 第 0 级 SE₀(0,4)：β(0)=000(MSB=0↑)，β(4)=001(MSB=0↑) → 同走上出 → 冲突
- 全部四个 SE 均冲突 → **位反转不能一次通过 Omega**

---

## 8. 再循环 Omega 网络

**概念**：输出反馈回输入再次路由，最多 $\log_2 N$ 次。N=8 时最多 2 次（共 6 级 > Benes 的 5 级 → 等效可重排无阻塞）。

**例子**：位反转 N=8，第 1 次全冲突 → 反馈再走第 2 次 → 可通过。

---

## 9. 解题速查

| 项目 | 公式 |
|:---|:---|
| 级数 | $\log_2 N$ |
| 每级 SE 数 | $N/2$ |
| 均匀洗牌 | 二进制循环左移一位 |
| SE 直通条件 | 上入控位=0，下入控位=1 |
| SE 交叉条件 | 上入控位=1，下入控位=0 |
| SE 冲突条件 | 两输入控位相同 |
| 再循环上限 | $\log_2 N$ 次 |
