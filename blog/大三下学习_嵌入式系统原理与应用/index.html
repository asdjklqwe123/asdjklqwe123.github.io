嵌入式微处理器 EMPU （Embedded Microprocesser Unit） = CPU + 硬件接口（存储器接口、键盘接口） + 辅助电路（时钟、A/D、D/A、调试电路等）（PS：AD转换器，将数字？逻辑转换为01） 嵌入式系统 = 嵌入式处理器 + 存储器 + 接口外围电路 + 外设 + 嵌入式操作系统 + 应用软件

全双工：可以同时读写 半双工：要么读要么写（如对讲机） 单工：只能接收（如广播给我们）

PTC P正的 T温度 C系数

  • 正温度系数
  • 说的是电路电阻 稳压二极管
  • 平时（5V）相当于断开
  • 瞬间电压超过5V时，瞬间导通（ns级） 这恒压二极管和电阻组合构成过压过流保护

  • b c e
