**任务计划程序**是 Windows 系统自带的一个管理工具，用于在**指定条件**下自动执行任务（比如启动程序、发送邮件、显示消息）。

其中发送邮件、显示消息已弃用

**核心特点**：
- **触发器灵活**：可以按“登录时”“开机时”“特定时间”“发生事件”等多种条件触发。
- **权限可控**：能设置**以最高权限运行**（解决普通启动方式下程序因权限不足而失败的问题）。
- **作用范围广**：可在用户登录前或系统启动早期就运行任务。

**使用流程**
1. **打开工具**：按 `Win + R`，输入 `taskschd.msc`，回车。
2. **创建任务**：点击右侧“创建基本任务”，填写名称（如“开机自启我的程序”）。
3. **设置触发器**：选择“当用户登录时”（常用）或“当计算机启动时”。
4. **设置操作**：选择“启动程序”，浏览并选中目标 `.exe` 文件。
5. **完成并设权限**：勾选“当单击‘完成’时，打开此任务的属性对话框”，点击完成。在属性窗口的“常规”选项卡中，勾选“使用最高权限运行”，然后确定。

这样，程序就会按设定的触发器自动运行了。
